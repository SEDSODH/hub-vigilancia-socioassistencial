> Hub institucional da Coordenação Estadual de Vigilância Socioassistencial. Centraliza normativos, materiais técnicos, capacitações e contatos para as equipes municipais de Vigilância Socioassistencial do Rio de Janeiro.

---

## Sobre

A **Coordenação Estadual de Vigilância Socioassistencial (COOVIG)** integra a Secretaria de Estado de Desenvolvimento Social e Direitos Humanos (SEDSODH). Nossa missão é apoiar os 92 municípios fluminenses na implantação e consolidação da Vigilância Socioassistencial — função essencial do SUAS dedicada a produzir, sistematizar e disseminar informações territorializadas para a gestão.

## Estrutura

| Seção | Conteúdo |
|---|---|
| **Início** | Missão, links rápidos para o Guia e materiais |
| **Normativos** | NOB/SUAS, LOAS, PNAS e documentos técnicos para consulta e download |
| **Portais** | Sistemas e portais oficiais do SUAS (CadSUAS, Censo SUAS, CadÚnico, etc.) |
| **SUAS em Dados** | Plataforma interativa desenvolvida pela COOVIG |
| **Materiais** | PDFs, cadernos técnicos e produções da equipe |
| **Vídeos** | Capacitações, oficinas e reuniões técnicas gravadas |
| **Equipe** | Membros da COOVIG e contato institucional |

## Executar localmente

```bash
git clone https://github.com/SEDSODH/hub-vigilancia-socioassistencial.git
cd hub-vigilancia-socioassistencial
npm install
npm run dev
```
